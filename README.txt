Замечательное приложение, позволяющее
читать случайные цитаты из аниме.
В качестве backend был использован следующий API:
https://animechan.vercel.app/

Использование:
- запустите приложение
- нажмите кнопку "get random quptes"
- выберите из списка аниме и персонажа
- нажмите на кнопку "show quote", чтобы показать цитату

Автор:
Росляков Дмитрий

Экраны:
- главный экран со списком цитат и кнопкой для получения случайных цитат
- экран с детальной информацией о цитате (аниме, автор, текст цитаты)

Стек технологий:
- Java
- RxJava2
- OkHttp3
- jackson
